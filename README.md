## Python版本3.12
## 安装所需库

```
pip install SpeechRecognition
pip install pydub
pip install python-telegram-bot
```

## 安装 FFmpeg

在 Windows 上，你可以下载 FFmpeg 并将其添加到系统的 PATH 环境变量中。

1. 从 [FFmpeg 官方网站](https://ffmpeg.org/download.html) 下载 FFmpeg。
2. 配置环境变量（自行查找具体步骤）。

## 使用说明

1. 在 resource 文件夹下放置录音文件。
2. 修改 `filename` 的值。
3. 运行程序即可得到识别结果。
